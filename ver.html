🎬 Auto Stream Loader

Carga automática de streams desde fuentes externas

📋 Cómo funciona:

  1. El sistema intentará cargar la página fuente en el iframe de abajo
  2. Desde ahí puedes navegar hasta el reproductor
  3. Una vez que el video esté reproduciéndose, usa las herramientas de desarrollador (F12) para inspeccionar el elemento del video
  4. Copia el enlace M3U8 y pégalo en el campo de abajo
  5. El sistema guardará el enlace para futuras visitas
